# Delivery Exception Dashboard

A simple client-side dashboard for managing logistics delivery exceptions.

## Project Structure

- `Delivery-Exception-Dashboard/`
  - `index.html` - Main dashboard page
  - `assets/images/` - Image assets used by the dashboard
  - `CSS/style.css` - Styles for the dashboard
  - `javascript/scripts.js` - Dashboard interactivity scripts

## Usage

Open `Delivery-Exception-Dashboard/index.html` in a web browser to view the dashboard.

## Notes

This is a static web project and does not require a build process or server to run locally.
